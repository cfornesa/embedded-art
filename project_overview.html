Embedded Art Platform

Node.js (Express) API with static frontends for Three.js, A-Frame, p5, and c2. The API lives in server/ and serves shared endpoints for all frontends.

API

server/src/routes/api.js – create, read, update, delete pieces.

Database

server/src/db.js – MySQL access and schema creation.

Security

reCAPTCHA v3 for create/delete, admin keys for edits.

Frontends

/threejs, /aframe, /p5, /c2
